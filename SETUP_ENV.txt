================================================================================
QUICK SETUP - Copy this to your .env file and add your Anthropic API key
================================================================================

Copy the section below into your .env file:

---START COPYING HERE---

# Anthropic API Key (REQUIRED - Get from https://console.anthropic.com/)
ANTHROPIC_API_KEY=

# Supabase Configuration (OPTIONAL - uses mock data by default)
NEXT_PUBLIC_SUPABASE_URL=https://placeholder.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=placeholder-key
SUPABASE_URL=https://placeholder.supabase.co
SUPABASE_ANON_KEY=placeholder-key

# Backend API URL
NEXT_PUBLIC_API_URL=http://localhost:8000

---STOP COPYING HERE---

================================================================================
STEPS:
================================================================================
1. Create .env file:
   touch .env

2. Open it in your editor:
   nano .env
   (or use VSCode, vim, etc.)

3. Paste the content above

4. Add your Anthropic API key after ANTHROPIC_API_KEY=

5. Save and close

6. Run the app:
   npm run dev

================================================================================
Getting Your Anthropic API Key:
================================================================================
1. Go to https://console.anthropic.com/
2. Sign up or log in
3. Navigate to "API Keys" in the left sidebar
4. Click "Create Key"
5. Copy the key (starts with sk-ant-)
6. Paste it after ANTHROPIC_API_KEY= in your .env file

Example:
ANTHROPIC_API_KEY=sk-ant-api03-xxxxxxxxxxxxxxxxxxxxx
